## English (US)

### Short Description (≤132 chars)
```
A beautiful, privacy-first new tab page with customizable widgets and no telemetry
```

### Detailed Description (~1000 chars)
```
Vantage replaces your new tab page with a beautiful, privacy-first dashboard featuring widgets for weather, RSS news feeds, todos, productivity tools, and more.

FEATURES:
• Animated background with real-time weather (sunrise/sunset, storms, clouds, rain)
• Multi-feed RSS reader + News aggregator (Atom + RSS, with DOMParser fallback)
• Widget variety: weather, news, todos, notes, quotes, clocks, countdowns, crypto prices, GitHub activity, Pomodoro timer
• Quick links with favicons and folder groups
• Customizable accent colors (Catppuccin, Tokyonight, Gruvbox, Nord, custom hex)
• Multiple workspaces (Home / Work / Hobby) — per-workspace widgets, themes, backgrounds
• Settings import/export, OPML feed import/export
• Geolocation auto-detect for weather or manual city entry

PRIVACY:
All your data stays on your device. No accounts, no telemetry, no tracking. Settings, feeds, todos, and preferences are stored locally in chrome.storage.local. Open-source at https://github.com/SysAdminDoc/Vantage.

KEYBOARD SHORTCUTS:
• / — Focus search
• Esc — Close modal
• Right-click — Quick menu (theme, background, settings)

ACCESSIBILITY:
WCAG 2.2 AA compliant: semantic HTML, ARIA labels, keyboard navigation, 4.5:1 color contrast, screen-reader tested (NVDA, VoiceOver).

Works on Chrome, Edge, Brave, Vivaldi, Opera, and Firefox.
```

