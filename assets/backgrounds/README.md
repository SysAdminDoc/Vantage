# Background Texture Assets

These raster textures are generated project assets for Vantage's animated background system.

They are procedural PNG overlays, not third-party photos:

- `atmosphere-grain.png` - shared haze and fine atmospheric grain.
- `mountain-atmosphere.png` - soft distant ridge and haze overlay.
- `ocean-surface.png` - ocean wave and foam texture.
- `lake-surface.png` - still-water reflection texture.
- `forest-canopy.png` - soft treeline/canopy texture.
- `dune-grain.png` - sand grain and dune contour texture.
- `icefield-grain.png` - frost, snow, and ice grain texture.

The textures are intentionally subtle. CSS gradients still drive the time-of-day palette, while these files add organic surface detail without external network requests or licensing requirements.
